## Auto-Delegation Patch

Problem found:

- Prompt-only rules in `SOUL.md` were not enough.
- A casual review request like "cek docs repo Naelvi_Studio, review aja ...
  jangan edit" was answered directly by Atlas using `cx/gpt-5.5`.

Fix added:

- File: `/opt/hermes-code/hermes-agent/gateway/run.py`
- A coding-instance gateway preprocessor detects review-only repository/docs
  requests and rewrites the message with a `ROUTING OVERRIDE`.
- The override requires first tool call to be `delegate_task`.
- It assigns exactly one Reviewer task using:
  - model: `cmc/deepseek/deepseek-v4-pro`
  - provider: `custom`
  - base_url: `http://127.0.0.1:20128/v1`
  - toolsets: `["file", "terminal"]`
  - read-only behavior.

Detection terms include review intent, repo/docs scope, and read-only wording:

- Review intent: `review`, `cek`, `audit`, `inspect`, `validate`, `verify`,
  `aman`, `aneh`, `kurang rapi`
- Scope: `repo`, `docs`, `Naelvi_Studio`, `repository`
- Read-only: `jangan edit`, `jangan ubah`, `review aja`, `cek doang`,
  `read-only`, `readonly`, `tanpa edit`

Backup file before patch:

- `/opt/hermes-code/hermes-agent/gateway/run.py.bak.review-route.*`
